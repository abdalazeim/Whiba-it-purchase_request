===============
Purchase Request
===============

Purchase Request Module for Odoo 17 - Whiba Holding

This module allows you to manage purchase requests within your organization.
Users can create, approve, and track purchase requests for materials and services.

Features
========

- Create purchase requests manually
- Automatic purchase request generation from procurement orders
- Multi-level approval workflow
- Request tracking and status management
- Integration with Purchase Orders (RFQ)
- Stock move allocation
- Analytics distribution support

Usage
=====

1. Go to *Purchase > Purchase Requests* to access the module
2. Create a new purchase request with required products
3. Submit the request for approval
4. Once approved, create RFQ from request lines

Configuration
=============

To enable automatic purchase request creation:

1. Go to a product form
2. In the *Inventory* tab, check *Purchase Request* along with the route *Buy*
3. Enable the *Replenish on Order (MTO)* route if needed
